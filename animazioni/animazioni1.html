    4 Animazioni Personalizzate

    üé® Creare Animazioni Personalizzate

    Quando le animazioni predefinite non bastano, puoi creare le tue usando CSS @keyframes. Ecco come fare passo dopo passo:

    Passo 1: Definire i Keyframes CSS

    Prima definisci l'animazione nel CSS:

    /* Nel tag <style> del tuo HTML */ @keyframes wiggle { 0%, 100% { transform: rotate(-3deg); } 50% { transform: rotate(3deg); } } @keyframes float { 0%, 100% { transform: translateY(0px); } 50% { transform: translateY(-10px); } } @keyframes glow { 0%, 100% { box-shadow: 0 0 5px #3b82f6; } 50% { box-shadow: 0 0 20px #3b82f6, 0 0 30px #3b82f6; } }

    Passo 2: Creare le Classi CSS

    Poi crea le classi che applicano l'animazione:

    /* Classi per applicare le animazioni */ .wiggle { animation: wiggle 1s ease-in-out infinite; } .float { animation: float 3s ease-in-out infinite; } .glow { animation: glow 2s ease-in-out infinite; }

    Passo 3: Usare nell'HTML

    Infine applica le classi agli elementi HTML:

    <!-- Esempi di utilizzo --> <div class="bg-orange-500 w-16 h-16 rounded-lg wiggle"></div> <div class="bg-teal-500 w-16 h-16 rounded-lg float"></div> <div class="bg-blue-500 w-16 h-16 rounded-lg glow"></div>

    Esempi in Azione

    Wiggle (Oscillazione)

    Perfetto per attirare l'attenzione

    Float (Galleggiamento)

    Effetto elegante e sottile

    Glow (Bagliore)

    Crea un effetto luminoso

    ‚öôÔ∏è Configurazione Tailwind (Opzionale)

    Se usi Tailwind in un progetto, puoi aggiungere le animazioni personalizzate alla configurazione:

    // tailwind.config.js module.exports = { theme: { extend: { animation: { 'wiggle': 'wiggle 1s ease-in-out infinite', 'float': 'float 3s ease-in-out infinite', 'glow': 'glow 2s ease-in-out infinite', }, keyframes: { wiggle: { '0%, 100%': { transform: 'rotate(-3deg)' }, '50%': { transform: 'rotate(3deg)' }, }, float: { '0%, 100%': { transform: 'translateY(0px)' }, '50%': { transform: 'translateY(-10px)' }, }, glow: { '0%, 100%': { boxShadow: '0 0 5px #3b82f6' }, '50%': { boxShadow: '0 0 20px #3b82f6, 0 0 30px #3b82f6' }, } } } } }

    Con questa configurazione puoi usare animate-wiggle, animate-float, e animate-glow come classi Tailwind normali!
